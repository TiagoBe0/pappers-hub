# pappers_html

Coloca aquí tus resúmenes en formato HTML con el patrón:

- `papper_id.html`

Opcionalmente, crea un archivo `index.json` con un arreglo de nombres para descubrimiento estable:

```json
[
  "papper_001.html",
  "papper_transformers_2025.html"
]
```

Si no existe `index.json`, la interfaz intentará leer el índice del directorio (`pappers_html/`) según soporte del servidor.
